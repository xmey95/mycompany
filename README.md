## Requirements

- `nodejs` (>=10.x)
- `npm` or `yarn`

run

```bash
npm install -g @angular/cli
```

## Running

#### First time

```bash
git clone https://github.com/xmey95/worderfulcompany.git
cd wonderfulcompany/frontend
yarn install
ng serve
```

#### Later on

```bash
cd wonderfulcompany/frontend
ng serve
```
